# L-Systems
Implementation of [L-Systems](https://en.wikipedia.org/wiki/L-system#External_links) using [raylib](https://www.raylib.com/)

The implementation is based on the wikipedia article (linked above) as well as on the [chapter](https://algorithmicbotany.org/papers/abop/abop-ch1.pdf) dedicated to L-Systems in the [book](https://algorithmicbotany.org/papers/abop/abop.pdf) "The Algorithmic Beauty of Plants", written by Prusinkiewicz and Lindenmayer himself.

Both the chapter and the book in question can be found at the provided links or in the `docs/books` subdirectory of this repository under the names `abop-ch1.pdf` (the chapter) and `abop.pdf` (the book).
